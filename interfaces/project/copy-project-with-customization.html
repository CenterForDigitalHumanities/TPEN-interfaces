--- title: Copy a Project permalink: /project/copy-project-options --- Copy a Project
  • Select the project you want to copy.
  • Choose the specific elements to include (e.g., metadata, group members, hotkeys, tools, and layers).
  • Optionally, select which members from the original project to include in the new one.
  • Decide how each layer should be copied: with annotations, without annotations, or excluded entirely. (By default, no layers are selected.)
  • Click "Copy Project" to create the new project with your chosen settings.
  • Review your newly copied project in the project list.
