Quick Guide to Create a New Project

Follow these steps to create a new project in TPEN.

Steps to Create a New Project From an Image :

  1. Click on the Create a New Project from My Actions tab on the home page.

  2. Click on the Transcribe button in the Transcribe Image tab.

  3. Enter the Project Name and a URL of the image you want to use for your project in the provided field.

  4. Click on the Create Project button to finalize the creation of your project.

  5. Your new project will now appear in the project list, and you can start transcribing it.

Steps to Import a Project from IIIF Manifest :

  1. Click on the Create a New Project from My Actions tab on the home page.

  2. Click on the Import Manifest button in the Import IIIF Manifest tab.

  3. Enter the IIIF Manifest URL in the provided field.

  4. Click on the Import Project button to finalize the creation of your project.

  5. Your new project will now appear in the project list, and you can start working on it.

Steps to Import a Project from T-PEN 2.8 :

  1. Click on the Create a New Project from My Actions tab on the home page.

  2. Click on the Import Project button in the Import T-PEN 2.8 Project tab.

  3. This will take you to the T-PEN 2.8 Login Page. Log in with your T-PEN 2.8 credentials.

  4. After successful login, you will see a list of your T-PEN 2.8 projects. Select the project you want to import.

  5. As you select a project, it will be imported into TPEN3, with all the project configurations and annotations preserved.

  6. All the collaborators from the T-PEN 2.8 project will be listed and you can invite them to collaborate on the imported project.

  7. Lastly, review the project details and you can start working on it.
