Detailed Guide to Create a New Project

Follow these steps to create a new project in TPEN.

Steps to Create a New Project From an Image :

  1. Click on the Create a New Project from My Actions tab on the home page.

  2. Click on the Transcribe button in the Transcribe Image tab.

  3. Enter the Project Name and a URL of the image you want to use for your project in the provided field.

  4. Click on the Create Project button to finalize the creation of your project.

  5. Your new project will now appear in the project list, and you can start transcribing it.

Steps to Create a New Project From an Existing Project :

  1. Click on the Copy an Existing Project from My Actions tab on the home page.

  2. Select the project you want to copy from the list of available projects.

  3. Enter a new Project Name for your copied project.

  4. Click on the Create Project button to finalize the creation of your copied project.

  5. Your new copied project will now appear in the project list, and you can start working on it.

Steps to Import a Project from IIIF Manifest :

  1. Click on the Import a IIIF Manifest from My Actions tab on the home page.

  2. Enter the IIIF Manifest URL in the provided field.

  3. Click on the Create Project button to finalize the creation of your project.

  4. Your new project will now appear in the project list, and you can start working on it.

Steps to Import a Project from T-PEN 2.8 :

  1. Click on the Import a T-PEN 2.8 Project from My Actions tab on the home page.

  2. Upload your T-PEN 2.8 project file in the provided field.

  3. Click on the Create Project button to finalize the creation of your project.

  4. Your new project will now appear in the project list, and you can start working on it.
